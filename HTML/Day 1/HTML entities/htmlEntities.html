HTML Entity Reference Guide

This guide lists common HTML entities, their codes, and how they appear in the browser.

Common Symbols

Less Than
Code: &lt; → Output: <
Greater Than
Code: &gt; → Output: >
Ampersand
Code: &amp; → Output: &
Non-breaking Space
Code: &nbsp; → Output: [ ] (space kept)

Legal & Currency Symbols

Copyright
Code: &copy; → Output: ©
Registered Trademark
Code: &reg; → Output: ®
Trademark
Code: &trade; → Output: ™
Cent
Code: &cent; → Output: ¢
Euro
Code: &euro; → Output: €

Arrows & Navigation

Right Arrow
Code: &rarr; → Output: →
Left Arrow
Code: &larr; → Output: ←
